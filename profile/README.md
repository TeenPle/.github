## 📱 서비스 소개

**TeenPle**은 고등학생이 자신의 학교를 인증하고 참여하는 **학교 전용 익명 커뮤니티 앱**입니다.
Flutter로 개발되어 **iOS & Android** 모두 지원하며, 학교 생활에 필요한 모든 것을 한곳에서 제공합니다.

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 🏫 **학교 인증** | 재학 중인 학교를 인증하고 해당 학교 커뮤니티 입장 |
| 📝 **익명 게시판** | 학교 친구들과 자유롭게 익명으로 소통 |
| 💬 **실시간 채팅** | 같은 학교 학생들과 실시간 채팅 |
| 🍱 **급식 정보** | 오늘의 급식 메뉴 한눈에 확인 |
| 📅 **시간표** | 나만의 시간표 관리 |
| 🎓 **졸업생 게시판** | 선배 졸업생들과의 소통 공간 |
